Project Overview

This project is a fully functional web application built using a modern full-stack setup.
The design is based on the Figma layout provided, and the goal is to keep the UI clean, responsive, and simple to use.
The backend is focused on stability, secure APIs, and smooth integration with the frontend.

Tech Stack

Frontend: React + Vite + TypeScript + Tailwind CSS
Backend: Node.js + Express + TypeScript
Database: PostgreSQL + Prisma ORM
Auth: JWT (Access + Refresh tokens)
